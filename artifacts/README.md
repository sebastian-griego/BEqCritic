# Artifacts

This folder contains experiment outputs that are useful for inspection/comparison but are not required to run the code.

- `beqplus/`: BEq+ (LeanInteract) evaluation outputs on ProofNetVerif (see `beqcritic.paper_pipeline.beq_plus_eval`).
- `proofnetverif/`: Selection outputs (JSONL) and related sweep artifacts for ProofNetVerif.
